First release. Five commands to read public ESPN sports data from
`site.api.espn.com` (no API key required).

## Commands

- `espn teams` - list all teams in a sport/league
- `espn scores` - current or recent scoreboard
- `espn news` - latest news articles
- `espn standings` - current league standings
- `espn schedule` - upcoming schedule (same scoreboard endpoint)

## Supported leagues

| Flag | Sport |
|------|-------|
| `nfl` | football |
| `mlb` | baseball |
| `nba` | basketball |
| `nhl` | hockey |
| `eng.1` | soccer (EPL) |

Default league is `nfl`. Pass `--league=nba` and `--sport` is inferred automatically.

## Output

All commands support `-o table|json|jsonl|csv|tsv`, `--fields`, `--template`,
and `-n` to limit results. Default output is a table.
